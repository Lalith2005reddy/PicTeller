# 🖼️ PicTeller: AI Image Captioning Tool

**PicTeller** is an AI-powered web application that generates smart and natural captions for your images. Built with a **React frontend**, **FastAPI backend**, and the **BLIP image captioning model**, it delivers fast and meaningful image descriptions.

---

## 🔥 Features

- 📤 Upload any image
- 🧠 Generate image captions using BLIP (Vision-Language Transformer)
- ✍️ Edit the generated captions
- 📋 Copy or share the caption easily
- ⚡ Fast, responsive UI

---

## 🛠️ Tech Stack

| Layer      | Tech                        |
|------------|-----------------------------|
| Frontend   | React.js                    |
| Backend    | FastAPI (Python)            |
| ML Model   | BLIP from Hugging Face 🤗    |
| Styling    | CSS                         |
| API Comm.  | Axios                       |
